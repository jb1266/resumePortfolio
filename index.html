Security-Focused Web Hosting with CI/CD (EC2)

Hosts a web app on EC2 instances in a private subnet, fronted by an Application Load Balancer in the public subnet with WAF and HTTPS via ACM. Self-Hosted VPN on AWS

Instead of paying for a commercial VPN that might log your data, built a personal VPN from scratch on AWS. Deployed an OpenVPN server inside a Docker container on an EC2 instance with a configured Security Group. The local machine connects via the OpenVPN client, routing all traffic through the EC2 instance — full control, zero third-party trust.
